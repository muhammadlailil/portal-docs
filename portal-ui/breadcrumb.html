Breadcrumb

Displays the path to the current resource using a hierarchy of links.

Home
Blog
Post 
<x-portal::breadcrumb>
    <x-portal::breadcrumb.item href="/">Home</x-portal::breadcrumb.item>
    <x-portal::breadcrumb.item href="/">Blog</x-portal::breadcrumb.item>
    <x-portal::breadcrumb.item>Post</x-portal::breadcrumb.item>
</x-portal::breadcrumb>

Publishing Component

If you want to customize the components, you can publish them by running the following command : 

php artisan vendor:publish --tag=portal-ui:breadcrumb --force
